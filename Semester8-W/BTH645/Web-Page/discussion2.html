Discussion on Web Usability using TEXT

1. Use only 2 to 3 fonts type through out the content and webpage.

2. Font Size must be consistent between pages and content information.

3. Use San Serif font for Heading or Title and Serif font for content information only.

4. Avoid using a paragraph that is too long, especially those are not even talking to the main topic.

5. Be concise about using font style such as Bold, Italic, Underline, ...

6. Use as much as 2 colours in text only. More colours usage will confuse user.

7. Section such as Email, Contact Us, Phone/Call,... should be clear and catch user attention.

8. Highlight important information where necessary and required. It is very important that user cannot miss any important information.

9. Enhance any piece of media use on the website, and make sure it is at least visible for normal user. Never attached a blurry image or an unplayable video.
