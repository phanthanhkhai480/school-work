Discussion Question

1) The reason of choosing the file format when creating your audio

The first file: .mp3 format because it is the most common file, it is lossy, compressed and reduced quality but it can be played easily anywhere. However, mp3 has different quality, known as kbps, can be range from 64kbps up to 320kbps.

The second file: .wav format because it is uncompressed file, it can be run on any Windows machine as it supported-by-default file for Windows audio. Moreover, this is the file type used to store media file in CD (Music Album) and require conversion to/from the CD. Since it is uncompressed, it is meant to be use for editing and creating music production.

The third file: .flac file format because the file type is used for lossless music data but also at the same time compressed any unnecessary data to reduce file size a bit. .flac lossless file type can be convert into any other lossless file type without losing any audio data. However, since this file type is not very popular due to its size, it's not very commonly used and support widely.
