Discussion on Web Usability using IMAGE

1. Pick an image type of very important, use the type that is not produce too low quality image or too much quality in image.

2. Use Raster/Bitmap image if user only need to glance at the image once or twice.

3. Use Vector image if that image will take user sometimes to see and understand its detail.

4. Provide subtitles for every image in case the image is unable to load.

5. Depends on the context, you can pick a heavy size image or a low size image for quality.

6. Choose image colour wisely, ensure that colour blind people don’t have a hard time to see.

7. Use image to represent a large pieces of text, because image get user to understand information better.

8. Don’t put too many image in one site, or too many image close together.

9. Be concise about image width, height, try to resize it so it can look good.

10. Never use image that has to vague meaning.
